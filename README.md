# Simulador de reforma

Juego de cartas tipo Reigns para el cumpleaños de Coral.

## Cómo jugar

```bash
npm install
npm run dev
```

Abre la URL en el móvil (o en el navegador). Desliza izquierda/derecha para decidir. Termina la reforma sin quedarte sin presupuesto ni cordura.

## Stack

Vite + React + TypeScript + Framer Motion
